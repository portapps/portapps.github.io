Installation

Download and install the latest portable setup where you want then run discord-portable.exe.

If you have already installed Discord™ from the original setup, move the following files :

  • %APPDATA%\discord\* to data

Then run discord-portable.exe and remove Discord™ from your computer.

Upgrade

For an upgrade, simply download and install the latest portable setup.

Modifications

Here is what differs from the original release to ensure portability :

  • Electron userData path is forced to data folder path.
  • Check for updates disabled (not for modules)
  • A shortcut is created at launch to allow native notifications and removed when Discord™ is closed

Configuration

Discord™ portable can be configured through the main YAML configuration file :

app:
  cleanup: true
  • cleanup : Cleanup leftover folders (default false)
